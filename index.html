Lumi Soft — WhatsApp Bot

Hai! Ini Lumi Soft — bot WhatsApp yang gue bangun pake Baileys, JavaScript/TypeScript, dan arsitektur based-plugin. Artinya: botnya modular, gampang dipasang plugin, dan ada fitur hot-reload supaya bisa update tanpa restart — cocok buat yang pengen bot on 24/7.

Tentang Lumi Soft

Bot WhatsApp modular, fokus ke produktivitas dan fun features.

Lumi Soft dibangun biar gampang dikembangkan — plugin-based, hot-reload, dan punya command handler yang rapi. Bisa dipake buat moderation, economy, roleplay, maupun automasi custom. Sistem plugin bikin kamu bisa nambah fitur tanpa ngubah core.

Spes singkat:
  • Runtime: Node.js + TypeScript
  • WA library: @whiskeysockets/baileys
  • Arsitektur: based-plugin (hot-reload)
  • Target: 24/7 uptime (deploy di VPS / PM2 / Docker)

Buat yang males restart tiap tambah fitur — hot-reload bakal otomatis me-load ulang module/plugin yang berubah. Jadi development loop lebih cepat, deployment lebih aman.

Cara deploy simple
// contoh pake PM2 + ts-node (development)
pm2 start src/index.ts --interpreter ts-node

// atau build & run
npm run build
pm2 start dist/index.js

Teknologi & contoh singkat

Potongan kode kecil biar kebayang cara kerja bot (TypeScript).

// contoh minimal: start Baileys + plugin loader
import makeWASocket from "@whiskeysockets/baileys";
import { loadPlugins } from "./plugin-loader";

async function start(){
  const sock = makeWASocket({ printQRInTerminal: true });
  const plugins = loadPlugins("./plugins"); // hot-reload-aware

  sock.ev.on("messages.upsert", async (m) => {
    const msg = m.messages[0];
    if(!msg.message) return;
    const text = msg.message.conversation || "";
    if(text.startsWith(".")){
      // dispatch to plugin handler
      await dispatchCommand(text.slice(1), msg, { sock, plugins });
    }
  });
}

start();
